# 💻 Web Development Internship — 3 Months Journey

A complete day-by-day record of my **3-month Web Development Internship**. Every single day is tracked — what I learned, what I built, and what assignments I completed. This repo is proof of consistent growth and daily practice over 90 days.

---

## 🎯 Purpose

- 📖 Track daily learning throughout the internship
- 💡 Document concepts, projects & assignments in one place
- 🚀 Show what is possible to learn in **3 months** of focused practice
- 🗂️ Serve as a reference for anyone starting their web dev journey

---

## 📁 Folder Structure

Every day has its own folder. Inside each day folder there are up to 3 subfolders depending on what happened that day:
```
web-dev-internship/
├── Day-01/
│   ├── learning/        # Notes and concepts learned
│   ├── project/         # Project built that day (if any)
│   └── assignment/      # Assignment given that day (if any)
├── Day-02/
│   ├── learning/
│   ├── project/
│   └── assignment/
├── Day-03/
│   ├── learning/
│   └── assignment/
│   ...
└── Day-90/
    ├── learning/
    └── project/
```

> **Note:** Not every day has a project or assignment folder. A day may only have a `learning/` folder if no project or assignment was given that day.

## 🗂️ What's Inside Each Folder

### 📖 `learning/`
Contains notes, code snippets, and concepts covered that day. This is the core of every day — what was taught and practiced.

### 🛠️ `project/`
Contains the complete code of any project built on that day. Projects are hands-on applications of the concepts learned.

### 📝 `assignment/`
Contains assignment solutions given by the instructor. Assignments are practice problems or mini tasks to reinforce the day's learning.
